Number Patterns

Do you like puzzles and mysteries? Then you are going to love algebra, because algebra is all about number patterns and finding numbers that are hiding. A pattern is a set of numbers that follow a rule. When you find the rule, you can tell exactly what number comes next. Let us become number detectives together.

Finding the Rule of a Pattern

Look at this row of numbers: 2, 4, 6, 8, … Each number is a little bigger than the one before it. From 2 to 4 we go up by 2. From 4 to 6 we go up by 2. From 6 to 8 we go up by 2 again. So the secret instruction, which we call the rule, is "add 2 each time." Once you know the rule, you can tell what comes next: after 8 we add 2 more, and we get 10.

Here is another pattern: 5, 10, 15, ▢. From 5 to 10 we add 5, and from 10 to 15 we add 5, so the rule is "add 5 each time." To find the missing number, we add 5 to the last number we can see: 15 + 5 = 20. This is just like counting five-dollar coins: 5, 10, 15, 20.

Patterns do not only go up. Some go down. In the pattern 20, 18, 16, 14, … the numbers get smaller by 2 each time, so the rule is "subtract 2 each time," and the next number is 14 − 2 = 12.

Think About It

  • A detective finds the rule in four steps: (1) Are the numbers getting bigger or smaller? (2) How much do they change by? (3) Does the same change happen each time? (4) Use the rule to find the missing number.
  • Patterns are everywhere — the days of the week, tiles on a floor, and the way you skip-count to 100.

Worked Example. Find the missing number in 3, 6, 9, ▢, 15. The numbers are getting bigger. From 3 to 6 the gap is 3, and from 6 to 9 the gap is 3, so the rule is "add 3 each time." After 9 we add 3 to get the missing number: 9 + 3 = 12. We check it works: 12 + 3 = 15. Yes! So the full pattern is 3, 6, 9, 12, 15.

Quick Check

Question: What is the rule for the pattern 10, 20, 30, 40, and what number comes next?
Answer: The rule is "add 10 each time," and the next number is 50.

Finding the Unknown

Now we come to the heart of our mystery: finding a number that is hiding. Look at this number sentence: 7 + ▢ = 12. The box ▢ is hiding a number, and we must find out which number it is. The sentence says, "7 plus some number equals 12," so we ask: what must I add to 7 to make 12? The hidden number is 5, because 7 + 5 = 12.

The Missing Number

There is a faster detective trick. Adding and subtracting are opposites, like opening and closing a door, so to find a missing number that is being added, you can subtract. For 7 + ▢ = 12, we work out 12 − 7 = 5. The hidden number is 5.

Worked Example. Find the missing number in ▢ + 6 = 15. Adding and subtracting are opposites, so we subtract: 15 − 6 = 9. We check it: 9 + 6 = 15. Yes! So ▢ = 9.

Worked Example. Find the missing number in 10 − ▢ = 4. We ask: what must I take away from 10 to be left with 4? Since 10 − 6 = 4, the hidden number is 6. So ▢ = 6.

Quick Check

  • What is the missing number in 8 + ▢ = 14? Work out 14 − 8 = 6, so ▢ = 6.
  • Mishka has 12 mangoes. She gives some away and has 8 left: 12 − ▢ = 8. Since 12 − 4 = 8, she gave away 4.

Remember

To find a number that is added: subtract to undo it.
Always check: put your answer back into the sentence to make sure it is true.

Letters and Boxes for Unknowns

We have used a box ▢ to stand for the hidden number. In algebra, we can also use a letter like n to do the very same job. When a box or a letter stands for an unknown number, we call it a variable. The sentences 5 + ▢ = 9 and 5 + n = 9 ask the very same question, and the answer is the same: 9 − 5 = 4, so ▢ = 4 or n = 4.

Worked Example (a money story). Kemar saves $30. Then he earns some more money, which we call n. Now he has $50 altogether. We write the story as 30 + n = 50. To find n, we subtract: 50 − 30 = 20. We check: 30 + 20 = 50. Yes! So Kemar earned $20.

Look Around You

  • When you save your pocket money, you are using algebra. "I have $15 saved and need $25 for a book" is 15 + n = 25, and the answer is n = 10.

True or False & If-Then

A good mathematician always asks: is a number sentence telling the truth? A number sentence is true when both sides are equal, and false when the two sides are not equal. The equals sign, =, is like a balance scale: for a sentence to be true, both sides must hold the same amount.

True or False Sentences

Let us test 6 + 3 = 9. The left side is 6 + 3 = 9, and the right side is 9. Both sides are 9, so they are equal. This sentence is true.

Now test 6 + 3 = 10. The left side is 6 + 3 = 9, but the right side says 10. Since 9 is not the same as 10, this sentence is false.

To test any sentence: (1) work out the left side, (2) work out the right side, (3) compare them. If they are equal, it is true; if not, it is false.

Worked Example. Is 4 + 5 = 8 true or false? Left side: 4 + 5 = 9. Right side: 8. Since 9 is not the same as 8, the sentence is false.

Worked Example. Is 12 − 4 = 8 true or false? Left side: 12 − 4 = 8. Right side: 8. Both are 8, so the sentence is true.

Quick Check

  • Is 7 + 2 = 9 true or false? It is true, because 7 + 2 = 9.
  • Is 10 − 3 = 6 true or false? It is false, because 10 − 3 = 7, not 6.

If-Then Puzzles

In an "If … then …" puzzle, someone tells you what a letter stands for, and you use it to finish the sentence. Putting a number in place of a letter is called substituting — it just means swapping the letter for its number.

Worked Example. If n = 40, then 90 − n = ? We are told n stands for 40, so 90 − n becomes 90 − 40 = 50.

Worked Example. If n = 28, then n + n = ? We swap n for 28, so n + n becomes 28 + 28 = 56. Notice that n + n is the same as adding n two times, which is the same as 2 × n. So 28 + 28 is the same as 2 × 28, which is also 56. This is how adding again and again turns into multiplying.

Worked Example. If n = 25, what is (a) 625 − n and (b) 135 − n? We swap n for 25 each time: (a) 625 − 25 = 600, and (b) 135 − 25 = 110.

Did You Know?

Substitute means swap the letter for the number you were given. Once only numbers are left, you work out the answer just as you always do.

Word Power

Here are the important words you met in this chapter. Each one is explained with an example.

Algebra Words to Know

Pattern: a set of numbers that follow a rule, in an orderly way. Example: 2, 4, 6, 8 keeps going up by 2.
Rule: the secret instruction that tells you how a pattern is made. Example: the rule for 5, 10, 15 is "add 5 each time."
Number sentence: a line of mathematics that uses numbers and signs. Example: 6 + 3 = 9 is a number sentence.
Variable (unknown): a box ▢ or a letter like n that stands for a number we do not know yet. Example: in 4 + ▢ = 10, the box is the unknown.
Equals (=): the sign that means "is the same as." Example: in 3 + 2 = 5, the equals sign tells us 3 + 2 is the same amount as 5.
True: a number sentence is true when both sides are equal. Example: 6 + 3 = 9 is true.
False: a number sentence is false when the two sides are not equal. Example: 6 + 3 = 10 is false.
Substitute: to swap a letter for the number it stands for. Example: if n = 40, then 90 − n becomes 90 − 40 = 50.
